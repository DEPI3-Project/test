A user-friendly, learnable, memorable WordPress admin dashboard theme.

== Description ==
Easy Admin Dashboard refreshes the WordPress admin with a clean, accessible design. It adds an options page to customize color scheme, accent color, compact menu spacing, and a custom logo for the login and admin bar.

== Installation ==
1. Download the zip release.
2. Upload via Plugins > Add New > Upload Plugin.
3. Activate the plugin.
4. Configure under Settings > Easy Admin.
